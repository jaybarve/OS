Installation process.

Simple. Clear. Step by step.

Desktop installation flow

1️⃣

Download the tool

Download the official Windows or Linux installation tool from this website.

2️⃣

Create bootable USB

Use Rufus, Ventoy, or Balena Etcher to create a bootable USB drive.

3️⃣

Boot from USB

Restart your computer and boot using the USB from BIOS or boot menu.

4️⃣

Install OS

Follow on-screen instructions to install Windows or Linux safely.

Mobile installation flow

1️⃣

Unlock & prepare

Enable USB debugging, unlock bootloader (if required), and install drivers.

2️⃣

Download firmware

Download official firmware or ROM for your Samsung, Xiaomi, or Nothing device.

3️⃣

Flash OS

Use Odin, Mi Flash Tool, or official Nothing installer to flash the OS.

4️⃣

Reboot & setup

Restart your device and complete the initial setup process.
